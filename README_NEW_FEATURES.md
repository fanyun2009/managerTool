# 任务管理工具 - 功能扩展版

基于原有的任务管理工具，增加了以下功能：

## 🆕 新增功能

### 1. 任务分类
- **分类选项**: 工作、学习、生活
- **显示位置**: 任务列表中的"分类"列
- **统计功能**: 实时显示各分类的任务数量和完成率

### 2. 优先级管理
- **优先级选项**: 高、中、低
- **颜色标识**: 
  - 🔴 高优先级 - 红色
  - 🟠 中优先级 - 橙色
  - ⚪ 低优先级 - 灰色
- **排序功能**: 支持按优先级升序/降序排列

### 3. 实时搜索
- **搜索框**: 位于界面顶部
- **实时过滤**: 输入内容时即时过滤任务列表
- **搜索范围**: 任务内容文本

### 4. 一键生成测试数据
- **生成按钮**: "生成测试数据"按钮
- **自定义选项**:
  - 生成数量（1-50个）
  - 是否清空现有数据
- **多样化数据**: 自动生成包含不同分类、优先级、完成状态的任务

## 📊 统计信息

界面底部显示实时统计信息，包括：
- 各分类的任务总数
- 各分类的完成率百分比
- 实时更新，随任务变化自动刷新

## 🔧 技术实现

### 数据结构扩展
```python
{
    "content": "任务内容",
    "completed": false,
    "category": "工作",      # 新增：分类字段
    "priority": "高"          # 新增：优先级字段
}
```

### 主要新增方法
- `update_statistics()`: 更新统计信息
- `on_search_change()`: 搜索内容变化处理
- `on_sort_change()`: 排序选项变化处理
- `generate_test_data()`: 生成测试数据对话框

## 🎯 使用说明

1. **添加任务**: 输入任务内容，选择分类和优先级，点击"添加"
2. **搜索任务**: 在搜索框中输入关键词，列表会自动过滤
3. **排序任务**: 使用排序下拉框选择排序方式
4. **查看统计**: 底部实时显示各分类统计信息
5. **生成测试数据**: 点击"生成测试数据"按钮，设置数量和清空选项

## 💾 数据持久化

- 任务数据保存在 `tasks.json` 文件中
- 新增字段会自动兼容旧版本数据
- 支持追加生成数据，不会覆盖现有任务
